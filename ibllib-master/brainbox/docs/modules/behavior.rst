`brainbox.behavior`: Functions for dealing with behavioral data
===============================================================

The behavior submodule of brainbox includes functions for dealing with various aspects of pure
behavioral data. Examples of behavioral data include reaction times, traces of appendages from
DeepLabCut, or pupilometry data.

Contains the `bb.behavior.wheel` submodule for dealing with experimental
wheel data.

At the moment, this module is empty. Let us know if you'd like to help expand it!
