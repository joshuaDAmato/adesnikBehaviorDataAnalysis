`brainbox.singlecell`: Tools for examining single cell activity
===============================================================

The `singlecell` module is a toolkit for examining the activity of single
units within a recording. At the moment we are limited to simple analyses
such as autocorrelations, but we aim to expand this
feature set in the future.
