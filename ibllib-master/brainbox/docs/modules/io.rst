`brainbox.io`: Functions for data I/O through Brainbox
===============================================================

The purpose of the `io` submodule is to provide utilities for reading in, exporting, and
transforming data via brainbox. This includes manipulation of matlab arrays of spikes into python
friendly format, and export into .mat files among other things.
